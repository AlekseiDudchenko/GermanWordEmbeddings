GermanWordEmbeddings

A toolkit to obtain and preprocess german corpora, train models and evaluate them with generated testsets

Download the whole project as a .zip file Download the whole project as a tar.gz file

Welcome

In my Bachelors thesis I trained german word embeddings with gensim's word2vec library and evaluated them with generated test sets. This page offers an overview about the project and download links for scripts, source and evaluation files
